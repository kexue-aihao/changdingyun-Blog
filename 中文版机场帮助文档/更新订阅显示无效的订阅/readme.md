# 更新订阅显示无效的订阅

------------------------

- 遇到这种情况，首先不要惊慌失措，并不是机场跑路了，而是机场更换订阅域名了，上官网重新复制一下订阅即可

- 只要机场方面没有说明自己跑路或者一些特殊情况的话，都不是账户被封禁造成的

- 如何获取最新的官网域名

> 1.加入官方的电报群

> 2.在官方存储库里获取最新的订阅域名(官网地址)
