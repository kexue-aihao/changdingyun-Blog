#Update subscription shows invalid subscription

--------------------------

- When encountering this situation, first of all, don’t panic. It’s not that the airport ran away, but that the airport changed the subscription domain name. Just go to the official website and copy the subscription again.

- As long as the airport does not explain that you ran away or there are some special circumstances, it is not caused by the account being banned.

- How to obtain the latest official website domain name

> 1. Join the official Telegram group

> 2. Obtain the latest subscription domain name (official website address) from the official repository
